# ⚙️ CareerPilot Backend

This backend powers the CareerPilot platform and provides APIs for authentication, resume processing, AI recommendations, and GitHub analytics.

Built with Node.js, Express, MongoDB, and Gemini AI.


## ✨ Features

### Authentication
- User Registration
- User Login
- JWT-Based Security

### Resume Processing
- PDF Resume Upload
- Resume Data Extraction
- Resume Analysis

### AI Career Guidance
- Gemini AI Integration
- Personalized Career Suggestions

### GitHub Analytics
- GitHub Profile Fetching
- Contribution Insights
- Repository Analysis


## 🛠 Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Multer
- Gemini AI API
- PDF Parse
