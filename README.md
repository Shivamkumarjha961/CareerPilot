# 🚀 CareerPilot

CareerPilot is a smart placement preparation platform designed for students who want to organize and improve their career journey in one place.

The platform helps students with:

- 📄 Resume Analysis
- 🤖 AI-Based Career Suggestions
- 📊 GitHub Profile Insights
- 💼 Job Application Tracking
- 🔐 Secure User Authentication
- 📈 Placement Preparation Management

## ✨ Features

### Resume Analysis
Upload your resume and receive useful insights to improve your profile.

### GitHub Analytics
Analyze your GitHub profile and understand your coding activity and contributions.

### AI Career Suggestions
Get personalized career recommendations powered by AI.

### Job Tracker
Track applied jobs, interviews, and application status in a simple dashboard.

### User Authentication
Secure registration and login system using JWT authentication.
