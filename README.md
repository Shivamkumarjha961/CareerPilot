# CareerPilot

CareerPilot is a smart platform built to help students improve their placement preparation in a simple and practical way. It combines resume analysis, GitHub profile insights, AI-based career suggestions, and job application tracking in one dashboard.

## Features

* Resume ATS score analysis
* GitHub profile evaluation
* AI career suggestions
* Job application tracker

## Tech Stack

* React
* Tailwind CSS
* Node.js
* Express.js
* MongoDB

## Project Structure

* Frontend_careerPilot → Frontend files
* Backend_careerPilot → Backend files

## Purpose

The main goal of this project is to support students during placement preparation by giving useful insights about their technical profile and helping them organize their progress.
