# 🎨 CareerPilot Frontend

This is the frontend of CareerPilot, built using React and Vite.

The application provides a clean and responsive interface where students can manage their placement preparation activities efficiently.

## ✨ Features

- User Registration & Login
- Resume Upload Interface
- ATS Score Display
- Resume History Tracking
- GitHub Profile Analysis
- AI Career Suggestions
- Job Tracking Dashboard
- Responsive Design

---

## 🛠 Technologies Used

- React.js
- Vite
- React Router DOM
- Axios
- Tailwind CSS
- Lucide React Icons
